# Zadanie

Приложение на C++/Qt (GUI) — пример проекта с использованием CMake.

## Описание
Проект содержит графическое приложение с окнами входа (vxod), регистрации (registr), главным меню (glavmenu) и главным окном (mainwindow). Интерфейсы описаны .ui-файлами, логика реализована в .cpp/.h.

## Возможности
- Окно входа
- Регистрация пользователя
- Главное меню и главное окно

## Зависимости
- Qt (5 или 6) — Qt Widgets
- CMake (>=3.5)
- Компилятор C++ (MSVC, MinGW или другой поддерживающий C++11+)

## Запуск
После успешной сборки запустить исполняемый файл из каталога build/Release (или соответствующего каталога конфигурации).
